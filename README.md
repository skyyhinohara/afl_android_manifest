# AfterlifeOS

## Getting Started ##
---------------
To get started with the AfterLife sources, you'll need to get
familiar with [Git and Repo](https://source.android.com/setup/build/downloading).

### Requirements
- Around 500GB disk space.
- Around 16GB RAM running Linux.

To initialize your local repository, use command:

```bash
repo init --depth=1 -u https://github.com/skyyhinohara/afl_android_manifest.git -b 14 --git-lfs
```

## Then sync up: ##

```bash
repo sync -c --no-clone-bundle --no-tags --optimized-fetch --prune --force-sync -j8
```

### Build our source ###

```bash
. build/envsetup.sh
goafterlife <device-codename>
```
